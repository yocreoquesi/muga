Welcome to MUGA.

URLs have become a mess. MUGA fixes that. Quietly, automatically, and honestly. Here's what you're getting.

Always active. No setup needed
✓
Strip 450+ tracking parameters: UTMs, fbclid, gclid, eBay noise, YouTube tokens… gone.
✓
Right-click → Copy clean link: share URLs without the garbage.
Affiliate support on by default
When you visit Amazon, Booking, eBay, etc. with no affiliate tag in the link, MUGA adds ours. You pay the same price. This is how the extension stays free.
Foreign affiliate notifications
Show a small toast when a link already has someone else's tag. Default is always "keep original". Replacing requires separate opt-in.
What MUGA will never do: silently replace someone else's affiliate tag (that's what Honey did), collect your browsing data, or send URLs to any server. The code is public. Read it.
🔒
100% local
Every URL is processed in your browser. Nothing is ever sent to a server.
⚙️
Opt-out any time
Turn off affiliate injection globally or per domain in Settings. You're always in control.
📖
Open source
MIT licensed. The code is public on GitHub. Read, fork, verify.
